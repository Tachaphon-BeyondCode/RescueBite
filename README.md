# RescueBite

**Rescue good food before it becomes waste.**

RescueBite is a real-time surplus food rescue platform that helps restaurants, cafes, bakeries, and supermarkets sell unsold food before closing time. Nearby customers can discover discounted rescue meals, claim them, and pick them up before expiry.

## Features

- **Nearby Deals Map** — Discover discounted surplus meals from local businesses
- **Real-time Claims** — Claim deals instantly with live inventory updates
- **Digital Pickup Tickets** — QR-ready tickets for seamless pickup
- **Merchant Dashboard** — Post rescue deals and track performance
- **Impact Hub** — Visualize sustainability impact and personal stats

## Tech Stack

- React 19 + Vite
- Tailwind CSS v4
- Framer Motion (animations)
- Lucide React (icons)
- LocalStorage persistence

## Getting Started

```bash
npm install
npm run dev
```

## Demo Flow

1. **Landing** → Emotional value proposition with impact metrics
2. **Nearby** → Browse deals on map, filter by category
3. **Detail** → View deal, check urgency, claim it
4. **Ticket** → Digital pickup ticket with QR placeholder
5. **Impact** → See your sustainability impact
6. **Merchant** → Dashboard with analytics and deal management
7. **Create Deal** → Publish a new rescue deal live
